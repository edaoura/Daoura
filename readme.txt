For the implemantation of the test automation i used:

1- Java as programming language
2- IntelliJ as IDE
3- Maven as build tool included in IntelliJ
4- TestNG as assertion framework.  This module is included with maven. normally It shouldn't be installed separately.
5- WebDriver as browser automation tool   (you can install it in windows over the cmd: C:\> choco install chromedriver)
choco o Chocolatey is a package manager for Windows,
